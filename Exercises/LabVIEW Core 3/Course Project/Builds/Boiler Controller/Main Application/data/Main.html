Main.vi

The boiler controller allows a user to start and shut down a boiler.

Use the Reset, start and Light pilot to sequencially Start the boiler.

Use the fuel control valve to change the flow of fulel into the boiler.

Use the Stop button to Shut down the boiler.

To shut down the system off ar in the event of emergency, use the emergency stop button to stop the application completely.

 Reset

Reset boiler.

Activate the run interlock on the boiler.

Start Sequence

Start primary fan to do a pre purge that blew off the combustible gas that accumulated in the boiler.

Pilot

Open the Pilot gas valve and ignite pilot flame simultaneously.

Main gas will be open when the pilot light proved.

Shutdown

Stop the flow of natural gas to the combustion chanber.

And stop combustion and do a post purge cycle.

Initiate Stop

Shutdown the application completely. Click this button to halt execustion of all parellel loops, close all open references, and exit.

Fuel Control Valve

Change the amount of fuel flowing into the boiler.

Until the boiler is running, this value remains 0. Once the boiler is running, the value may range from 10-75%.

Status

Shows the tatus of boiler when the boiler going through different stages of its operation.

Status shows appropriate message for the all of the boiler steps like Lock out, prepurge, Iginition, running, Stop, Post purge, ets......

Primany Fan

Primary fan runs during purge cycle to clear any cumbustaible gas that accumulated in the system.

Pilot Gas Valve

Pilot gas valve opens in the iginition stage of boiler which allows fuel that helps to make a pilot flame.

Pilot LED

Pilot flame is ignited simultaneous to the pilot fuel start flowing to the pilot line.

Forced Draft Fan

Forced draft fan helps to allow air into the combustion chamber for the process of combustion, Which start as the boiler start operating and stops when boiler stops.s
